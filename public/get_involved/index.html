The data

The data for this project is from the Avon Longitudinal study of Parents and Children (ALSPAC, also known as Children of the 90s). For more information about the study visit http://www.bristol.ac.uk/alspac/.

As the ALSPAC data is highly sensitive we cannot open the data to the public without ensuring the data is anonymised and non-identifiable. To achieve this an anonymised (synthetic) dataset will be generated using the R package SynthPop (Nowok, Raab & Dibben, 2016) using a non-parameteric method. The missing values will not be imputed you may deal with them as you see fit. The dataset contains approximatley 15,000 observations of over 80 variables. Along with the sythetic data we will release a description of the annonymisation procedure and a summary of comparisons between the Sythetic and Real Datasets.
